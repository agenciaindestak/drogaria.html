# PROMPTS PARA QWEN - LANDING PAGE DROGARIA MACEDO

## Prompt Principal (para geração completa)
```
Crie uma landing page de alta conversão para a Drogaria Macedo com:
- Header com logo, navegação e botão CTA WhatsApp
- Hero section com benefícios: entrega grátis acima R$99, mínimo R$50 para entrega via motoboy, farmacêutico de plantão
- Seção de serviços: medicamentos, aplicação de injetáveis, aferição de pressão/temperatura, manipulação, orientação farmacêutica
- Seção de entrega com detalhes de valores e condições
- Seção farmacêutico de plantão com diferenciais
- Depoimentos de clientes
- CTA final com WhatsApp e telefone
- Footer com contato, horário e links
- Botão flutuante WhatsApp
- Design responsivo, cores azul (#1E88E5), verde (#43A047), laranja (#FF6F00)
- Fonte Poppins e Open Sans
- Animações CSS (float, hover effects, smooth scroll)
- Imagens do Unsplash relacionadas a farmácia
```

## Versão Recomendada: Qwen 2.5 (mais recente)
- **Qwen 2.5 72B** - Para geração completa de código
- **Qwen 2.5 32B** - Para código mais leve
- **Qwen 2.5 14B** - Versão balanceada

## Alternativas caso Qwen não funcione:
- **Claude 3.5 Sonnet** - Excelente para código HTML/CSS
- **GPT-4** - boa opção para estrutura completa
- **Gemini 2.5** - boa para código moderno
